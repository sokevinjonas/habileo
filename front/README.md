# 📱 Habileo Frontend

Habileo est une application mobile d’essayage virtuel basée sur l’intelligence artificielle.  
Elle permet aux utilisateurs de visualiser une tenue sur eux avant de l’acheter ou de la faire confectionner.

Ce frontend est développé avec Ionic (Angular) et communique avec une API backend Flask.

---

## 🚀 Fonctionnalités

- Upload de la photo utilisateur 👤  
- Upload d’une tenue 👕  
- Génération de rendu via IA 🤖  
- Comparaison avant / après 🖼️  
- Interface mobile intuitive 📱  
- Expérience fluide et rapide ⚡  

---

## 🧱 Stack technique

- Ionic ⚡  
- Angular 🅰️  
- TypeScript 📘  
- HTML / SCSS 🎨  
- API REST (backend Flask)
